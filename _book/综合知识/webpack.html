webpack 安装介绍

webpack 的主要的作用是为了打包所有的资源、图片、样式等等，当 webpack 处理应用时，会递归的调用一个依赖图，然后将这些模块尽心打包。

在安装一个要打包到生产环境的安装包时，你应该使用 npm install --save，如果你在安装一个用于开发环境的安装包（例如，linter, 测试库等），你应该使用 npm install --save-dev。请在 npm 文档 中查找更多信息。
